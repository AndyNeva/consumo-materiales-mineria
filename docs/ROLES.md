# 🗄️ ROL: BASE DE DATOS (SQLite)
📆 Duración: 1 mes + 2 semanas (6 semanas)
🎯 Objetivo final:
Tener una base de datos clara, estable y fácil de usar para el backend y el equipo de datos.
⚠️ Regla de este rol:
•	❌ No tocar HTML
•	❌ No tocar JavaScript
•	❌ No entrenar modelos
✔️ Solo manejar datos guardados
________________________________________

🎯 Objetivo
Entender qué información necesita el sistema y cómo se relaciona.

•	Preguntarse:
o	¿Qué datos entran?
o	¿Qué datos se consultan?
o	¿Qué datos se predicen?

•	Sistema → guarda consumo

o	Backend
